자바 프로그램의 패키지
==> 현대 자바프로그램은 프로그램을 개발할 때
    디자인 패턴을 적용시켜서 프로그램을 제작한다.

    자바에서는 MVC 패턴을 이용해서 프로그램을 제작하는 것이 일반적이다.

    M   - Model(데이터, 데이터가공)
    V   - View(화면, 유저인터페이스)
    C   - Controller

    우리가 제작하는 프로그램에서 패키지 구조는

        db
        Controller
        Dao
        vo
        sql
        [ view ]

    로 구분이 되는데
    각 패키지에 추가되는 클래스들은 각자의 임무에만 충실하게 코딩이 되어야 된다.
